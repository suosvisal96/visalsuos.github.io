1.0.2

The maturity release

Latest · Production

Full mode coverage, native input, and a robust composer pipeline.

✦ New modes & flows
  • Image-to-image mode with three pathways:
    • New Image flow
    • Edit Image flow
    • Previous-image carryover support
  • Ingredients-to-video smoke flow
    • New ingredients-cat test data
    • Test prompts and reference images
  • Full smoke-test coverage across every mode:
    • Text to Image — New Image & Edit Image
    • Text to Video — 6s & Concat 10s
    • Frame to Video
    • Ingredients to Video
    • Image to Image — New & Edit
⌥ Native input helper
  • Real paste, Enter, Escape and coordinate clicks via Windows-native helper
  • New runtime input mode setting: native or debugger
✓ Composer & submit fixes
  • Closes the Speed/Quality/Aspect Ratio sheet before submit on Edge
  • Prevents settings or aspect-ratio controls from being clicked as Send
↻ Concat video flow
  • Captures the previous video's last frame for chained generations
  • Uploads only one carryover frame per concat hop
  • Retries concat prompts with a clean composer state
  • Prevents duplicate thumbnails on retry
◐ Image generation controls
  • New Speed/Quality image-model setting
  • Dedicated image-quality setting
  • Tighter aspect-ratio selection
◈ Result handling
  • Better fallback when grabbing image cards
  • Visible-crop fallback when Grok does not expose a direct image URL
  • More reliable detection of generated cards
⚠ Rate-limit handling
  • Detects Grok's 720p downgrade message
  • Avoids treating 720p-to-480p downgrades as fatal rate-limit errors
+ Behind the scenes
  • Native Windows input helper for more reliable typing
  • Account & usage tracking foundation
  • Build-flavor metadata for cleaner version reporting
  • Expanded internal test coverage across every mode
